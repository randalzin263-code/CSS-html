# CSS-html Pagina de teste

Esse projeto é uma pagina de teste para um site

Neste projeto foi utilizado o HTML

Após baixar a pasta do arquivo, abra o visual studio clique na opcão aquivo e em seguida clique em abrir pasta e escolha a pasta do CSS-html, clique em " Sim, confio nos autores ", para ver o site rodando clique em "Go live" na parte inferior esquerda.
